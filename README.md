# Smart-fridge-SCADA
Smart Fridge IoT with ESP32 – monitors temperature and humidity, controls compressor, and publishes data via MQTT.


Smart Fridge ESP32 IoT Project

Проєкт демонструє роботу «розумного холодильника» на базі мікроконтролера ESP32. Використовуються датчики DHT22 для вимірювання температури та вологості, а сервомотор SG90 імітує роботу компресора. Дані відображаються на LCD-дисплеї та передаються через MQTT у вигляді JSON-повідомлень для подальшої візуалізації або моніторингу.

Особливості:

Зчитування температури та вологості в реальному часі.

Керування компресором за допомогою сервомотора.

Відображення стану системи на LCD-дисплеї.

Передача даних через MQTT (температура, вологість, стан компресора).

Легко розширюється для інтеграції з IoT-платформами або SCADA-системами.
